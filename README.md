# 🧠 Emotion Detection & Mental Health Analysis using Machine Learning

## 📌 Overview

This project focuses on detecting human emotions from text data and analyzing their relationship with mental health conditions using Machine Learning and Deep Learning models.

It classifies emotions such as:

* Joy 😊
* Sadness 😢
* Anger 😡
* Fear 😨

and provides insights into possible mental health risks.

---

## 🚀 Features

* Text preprocessing (cleaning, tokenization, lemmatization)
* Multiple ML models:

  * Logistic Regression
  * Random Forest
  * Naïve Bayes
  * LSTM (Deep Learning)
* Performance evaluation:

  * Accuracy
  * Precision, Recall, F1-score
  * Confusion Matrix
* Mental health correlation analysis

---

## 🛠️ Tech Stack

* Python
* scikit-learn
* TensorFlow / Keras
* NLTK
* Pandas, NumPy, Matplotlib, Seaborn

---

## 📊 Results

* Achieved **95.5% accuracy**
* LSTM outperformed traditional ML models
* Strong correlation found between:

  * Sadness → Depression
  * Anger → Anxiety indicators
